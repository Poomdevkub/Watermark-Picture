Steps for Applying a Watermark Using DWT:

Discrete Wavelet Transform (DWT) is a mathematical technique used for image processing, which allows the decomposition of an image into different frequency sub-bands. This enables us to embed watermarks in specific sub-bands, making the watermark more resilient to various image manipulations. DWT helps preserve the quality of the image while maintaining the integrity of the watermark.

  1. Select the original image file:
  2. Choose the image you want to use as the base for embedding the watermark.
  3. Select the watermark image file:
    • Choose the image that you wish to embed as a watermark.
  4. Adjust the Alpha value (from 0.01 to 0.9):
    • Alpha represents the strength or visibility of the watermark.
    • A lower alpha value makes the watermark more transparent while a higher value makes it more visible.
  5. Upload and wait for the system to process:
    • After selecting both images and adjusting the alpha value, upload the images and wait for the system to generate the watermarked image.
  6. Download the processed image:
    • Once the watermark is successfully embedded, you can download the final image.
